Step 1: Tell Us Your Goals

Clean input form: budget, risk tolerance, sectors, time horizon, and optional notes. No jargon. One button: "Get Recommendation."

Budget: $100,000
Risk: Moderate
Sectors: AI, Clean Energy
Horizon: 5 years

Step 2: Watch Agents Work

Friendly progress messages ("Researching market conditions...") with an agent transparency panel. Users can inspect each agent's role, skill file, prompt, and live output.

Transparency Layer
• SVG flow diagram (live animation)
• Agent sidebar with kill/restart
• Skill files and prompt templates
• Real-time output per agent

Step 3: Your Recommendation

Summary, holdings table, sector donut chart, risks and mitigations, verification checks, disclaimer. Accept or request changes (triggers revision loop).

User Actions
• Accept Recommendation
• Request Changes (re-runs loop)
• View Agent Details (expandable)
Why Transparency Matters

Users trust recommendations more when they can see how agents reached them. The "View Agent Details" section shows each agent's skill, prompt, and output. Nothing is hidden.
