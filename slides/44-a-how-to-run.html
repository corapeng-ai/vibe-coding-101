How to Run It

One command in Claude Code launches the full multi-agent pipeline

Invoke the Workflow
Workflow({ scriptPath: 'investment-advisor.js', args: { query: "I have $100k to invest. Moderate risk. Interested in AI and clean energy. 5-year horizon." } })
What Happens
  • 1
    Intake agent parses your query — extracts budget, risk, sectors, horizon into typed JSON (~10s)
  • 2
    4 research agents fan out in parallel — each searches the web for live market data (~40s)
  • 3
    Portfolio builder synthesizes all research — produces 10 holdings with weights and rationale (~20s)
  • 4
    3 skeptics verify adversarially — flags get auto-fixed, max 2 revision rounds (~30s)
  • 5
    Final portfolio returned with full audit trail — profile, research, portfolio, verdicts, metadata

Total time: ~90 seconds — vs 2-3 hours manually
