Agents 2-5

Research Phase: 4 Agents in Parallel

Each specialist searches the live web and returns structured findings

Market Research Agent

Web search: macro trends, sector outlook

Searches for current GDP, interest rate environment, sector-specific growth drivers and headwinds. Rates each sector's trend (strong growth / stable / declining) with confidence level.

Output: outlook, sectors[], macroFactors[], risks[]

Stock Screener Agent

Web search: equities by fundamentals

Screens for P/E < 35, revenue growth > 10%, market cap > $5B. Adjusts for sector norms. Returns 12-15 real ticker candidates with approximate valuations and one-line rationale each.

Output: candidates[{ticker, sector, peRatio, growth, reason}]

Risk Analyst Agent

Web search: volatility, correlation data

Assesses overall risk level, identifies specific risk factors with severity ratings. Analyzes correlation between target sectors. Estimates worst-case drawdown scenario.

Output: overallRisk, factors[], correlationNotes, maxDrawdown

News Sentiment Agent

Web search: headlines, earnings, red flags

Scans news from the last 30 days. Assesses sentiment per sector with top headlines. Flags red flags: pending earnings, regulatory changes, insider activity, supply chain issues.

Output: overallSentiment, sectorSentiment[], redFlags[]

All 4 run simultaneously via parallel() — wall-clock time = slowest agent (~40s), not sum (160s)
